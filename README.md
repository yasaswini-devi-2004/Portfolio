This is a portfolio website,consisting of mutiple pages like home,feedback,profile and cv links.
The tech stack used is HTML,Tailwind css,React and javascript in frontend and node,express and postgresql in backend.
It is both website and mobile compatible with easy user navigation and ease of use.
Home page renders the home page of the website with my photo 
Profile pages allows you to send email to me,view my linkedin and github accounts
Feedback page allows you to rate my website,enter your name and give comments to my website which will be stored in my database.My database stores user name,rating,their comments and the time they submitted the feedback
cv page allows you to view and also download my resume 
this is a full stack website powered by pern stack containing a powerful frontend and backend
